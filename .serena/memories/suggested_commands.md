# Suggested Commands

## Code Quality
- **Analyze:** `flutter analyze`
- **Format:** `flutter format .`
- **Fix:** `dart fix --apply`

## Testing
- **Run all tests:** `flutter test`
- **Run specific test file:** `flutter test path/to/test_file.dart`

## Dependency Management
- **Get packages:** `flutter pub get`
- **Update packages:** `flutter pub upgrade`

## Build
- **Clean:** `flutter clean`
- **Build runner (if used):** `flutter pub run build_runner build --delete-conflicting-outputs`
