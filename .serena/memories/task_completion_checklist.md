# Task Completion Checklist

Before finishing a task, ensure:
1. [ ] Code follows project architecture and conventions.
2. [ ] All imports are clean and necessary.
3. [ ] `flutter format .` has been run.
4. [ ] `flutter analyze` passes without errors or warnings.
5. [ ] Relevant tests have been executed and passed (`flutter test`).
6. [ ] Documentation (PRD, Spec, or Bug Report) is updated if necessary.
7. [ ] Commits follow the convention described in `documentation/guidelines/developement-guidelines.md`.
